# Задача №37: объединение в одну коллекцию
Есть два массива строк. Надо их объединить в одну коллекцию, исключив повторения, не используя Linq. Пример:
```
{"1", "2", "1"} + {"3", "2"} => {"1", "2", "3"}
```

## Build

Build:
```
dotnet build
```

Just run:
```
dotnet run --project src\App.csproj
```
